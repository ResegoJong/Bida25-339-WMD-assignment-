# BAC Navigation Website

## Project Overview
This website was created as a school project to help students and visitors navigate BAC campuses easily. It provides information about campus locations, directions, transport schedules, and important facilities.

## Pages Included
- Home Page (index.html) – Introduction to the website
- About Page (about.html) – Information about BAC and the purpose of the website
- Locations Page (locations.html) – List of BAC campuses and transport schedules
- Directions Page (directions.html) – Step-by-step directions around campus
- Contact Page (contact.html) – Student communication system using Outlook-style format

## Features
- Campus location tables
- Transport timetable
- Campus facility opening hours
- Step-by-step navigation guide
- Student communication form

## Purpose
The purpose of this project is to make it easier for new students to understand and navigate BAC campuses and services.
